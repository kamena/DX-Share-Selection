== Description ==

WP Selected Text Sharer is a plugin, which allows to share or search the selected texts / words of yout site. A small popup is displayed above the selected text which contains links to share / search selected text.

This plugin requires a 4 kb [jQuery plugin](http://www.aakashweb.com/jquery-plugins/selected-text-sharer/) or 5 kb [Javascript](http://www.aakashweb.com/javascripts/selected-text-sharer) to display the the popup containing the share/search sites.

= What is the use ? =

* When a user selects a "Quote" of your site, a popup is displayed. When that quote is tweeted, the selected Quote and the link to that blog post is also tweeted in Twitter. Similarly other sites also act. Hence increasing site visibility.

* Search engines (Your site's search / Google adsense search / Yahoo search etc ...) can also be added to the popup. When the selected text is searched, your sites usage is increased and also Google adsense search is utilised.

* Apart from sharing texts only, this plugin acts like a social bookmarking button also.

= Features =

* Light weight, 4 kb jquery plugin is required. Also ordinary javascript version (5 kb) is also available to display the popup.
* Fully customizable. Change the colors of the popup completely to suite the site.
* Add upto 96 social bookmark links to the popup, with icons.
* Cross-Browser support.
* Automatic selected text truncation for Twitter.
* Shorten the post link using Bitly.
* Available in German and Chinese languages also.
* Alternative to usually used social-bookmarking buttons.
* Load the scripts either in header or footer.
* Simple and light interface.
* Template system for adding links to the popup.
* No knowledge of coding required.
* Integration with [WP Socializer plugin](http://www.aakashweb.com/wordpress-plugins/wp-socializer/)

== Installation ==

1. Upload `wp-selected-text-sharer` to the `/wp-content/plugins/` directory.
1. Activate the plugin through the 'Plugins' menu in WordPress.
1. Customize the plugin in the WP Selected Sharer admin page (Settings -> WP Selected Sharer).
1. Enjoy !
